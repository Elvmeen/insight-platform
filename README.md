# NAPS UNIJOS — Department of Physics

Frontend for the National Association of Physics Students (NAPS), University of Jos.

## Stack

- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- React Router

## Local development

```sh
npm install
npm run dev
```

Build:

```sh
npm run build
```

## Backend

The backend is a separate Django project that exposes a JSON API consumed by this frontend. See `BACKEND_BRIEF.md` for the contract and integration points.
